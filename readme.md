🌿 Website Virtual Gift - Green Edition

Selamat datang di repositori Website!
Ini adalah proyek website personal interaktif yang mengusung tema "Green Edition" dengan palet warna bernuansa alam (mint, emerald, forest green).
Website ini dirancang dengan antarmuka yang modern, animasi yang mulus, dan beberapa fitur interaktif kejutan.

✨ Fitur Utama

- Tema Hijau Segar (Green Edition):
  Desain visual yang memanjakan mata dengan kombinasi warna hijau yang konsisten di seluruh halaman (#52b788, #b7e4c7, #071a0f).

- Animasi Partikel Melayang:
  Efek floating particles (berbentuk daun, bintang, dan berlian) yang menghiasi latar belakang website.

- Galeri Bento Grid:
  Menampilkan foto atau karya dalam tata letak grid bergaya bento yang responsif, lengkap dengan fitur lightbox untuk memperbesar gambar.

- Interactive Quiz Overlay:
  Fitur kuis tersembunyi yang menantang pengunjung dengan soal matematika "ilusi" tingkat dewa.

- Animasi Penutup (Confetti):
  Tombol interaktif di bagian penutup yang memicu animasi confetti saat diklik.

- Custom Loader:
  Layar loading dengan animasi cincin berputar (spinning rings) sebelum halaman utama dimuat sepenuhnya.

- Responsive Mobile Navbar:
  Menu navigasi yang otomatis beradaptasi dengan efek blur (glassmorphism) saat di-scroll di perangkat seluler.

🚀 Cara Menjalankan Secara Lokal

Karena ini adalah website statis, kamu tidak perlu menginstal dependensi server backend seperti Node.js atau Python.

Langkah-langkah:
1. Clone atau download repositori ini ke komputermu.
2. Buka folder proyek tersebut.
3. Klik dua kali file index.html untuk membukanya di browser (Chrome, Firefox, Safari, dll).

(Opsional tapi disarankan)
Gunakan ekstensi seperti Live Server di Visual Studio Code agar browser otomatis reload saat ada perubahan file.
